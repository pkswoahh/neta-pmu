# Neta

Herramienta de gestión para micropigmentadoras independientes.
Tagline: "Tu negocio, claro como el agua."
Precio: $15 USD/mes. Marca: Marketing de Roberto.

## Stack
React + TypeScript + Vite · Supabase · Tailwind CSS · Netlify

## Público
Micropigmentadoras independientes en LATAM y USA. 
Usan principalmente el celular.

## Módulos
1. Dashboard — resumen del negocio
2. Procedimientos — registro diario de servicios
3. Gastos — registro de egresos
4. Configuración — opciones personalizadas por usuaria
